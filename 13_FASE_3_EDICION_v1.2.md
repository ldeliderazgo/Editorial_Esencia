PASO 3: NIVEL 1 — EDICIÓN DE CONTENIDO

Basándome en mi feedback a tu resumen, comienza NIVEL 1: edición de contenido.

NIVEL 1 cubre:
✓ Análisis estructural: ¿el orden de capítulos es lógico?
✓ Análisis de argumentos: ¿cada argumento tiene premisa→desarrollo→evidencia?
✓ Análisis de ejemplos: ¿son concretos, específicos, relevantes?
✓ Redundancias: ¿hay ideas que se repiten sin justificación?
✓ Saltos lógicos: ¿faltan pasos intermedios en el razonamiento?
✓ Promesas vs. cumplimiento: ¿la intro promete X? ¿el libro entrega X?
✓ Consistencia de tono con Capítulo 0: ¿hay secciones que suenan a otro libro?

QUÉ HACER:
→ Abre Track Changes en Word
→ Por cada cambio mayor: comentario explícito explicando por qué
→ No reescribas párrafos enteros (máximo sugerencias puntuales)
→ Si un capítulo debería eliminarse o reordenarse: comunícalo con comentario, no lo hagas
→ Si el tono se desvía del Capítulo 0, señálalo con un comentario específico

PASO 5: NIVEL 2 — EDICIÓN DE ESTILO

NIVEL 2 cubre:
✓ Voz narrativa: ¿se usa la segunda persona (tú) consistentemente?
✓ Consistencia con Capítulo 0: ¿el tono global coincide con la referencia?
✓ Párrafos: ¿están en rango 3-6 líneas (75-150 palabras)?
✓ Oraciones: ¿están en rango 15-25 palabras (máximo 35)?
✓ Voz activa: ¿hay pasivas evitables?
✓ Flujo: ¿los párrafos conectan lógicamente?
✓ Ejemplos: ¿cada uno ilumina el argumento?
✓ Repeticiones: ¿hay estructura paralela consistente?

PRINCIPIO FUNDAMENTAL:
Valora más la voz original del autor que la perfección gramatical. Un párrafo con personalidad y una coma discutible es preferible a un párrafo anodino y técnicamente perfecto. Solo sugiere cambios cuando la claridad o la coherencia con el Manual de Estilo se vean seriamente comprometidas.

QUÉ HACER:
→ Trabajar sobre el manuscrito post-Nivel 1 (cambios de contenido ya resueltos)
→ Track Changes activado
→ Refrases puntuales para mejorar claridad (máximo 1-2 oraciones por cambio)
→ Si un párrafo debería reescribirse completamente: comentario sugiriendo qué cambiar, no la reescritura completa
→ Marcar párrafos que suenan como autoayuda vacía
→ Comparar constantemente con el Capítulo 0

PASO 7: NIVEL 3 — CORRECCIÓN ORTOTIPOGRÁFICA

NIVEL 3 cubre (basado en 02_REFERENCIA_MANUAL_ESTILO.md):

✓ Comillas: « » para citas, " " para términos, ' ' para cita dentro de cita
✓ Cursivas: extranjerismos, títulos, términos técnicos (SOLO)
✓ Negritas: conceptos definidos explícitamente (máximo 3-4 por capítulo)
✓ Números: letra hasta 10, cifra de 11+, siempre cifra para %, años, precios
✓ Puntuación: contra normas del Manual (coma, punto y coma, dos puntos, raya)
✓ Mayúsculas: según RAE (días/meses minúscula, instituciones mayúscula)
✓ Extranjerismos: eliminar comodines (sinergia, paradigma, disruptivo)
✓ Espacios: único entre oraciones, sin espacios antes de puntuación
✓ Formato: márgenes, sangría, interlineado consistentes

NOTA SOBRE SEPARACIÓN DE ROLES:
